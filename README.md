# Disease Outbreak Prediction using ARIMA

This project predicts disease outbreak trends using time-series forecasting on COVID-19 datasets.

## Files Included
- `app.py` → Streamlit dashboard
- `disease_outbreak_analysis.ipynb` → model training and analysis
- `arima_outbreak_model.pkl` → saved trained model
- `CONVENIENT_global_confirmed_cases.csv` → dataset

## Live Demo
https://diseasepredictionappgit-jrdpgmmg3ujubmltqsm7af.streamlit.app/

## Features
- ARIMA forecasting
- Streamlit dashboard
- Time-series analysis
- 14-day future prediction

## Technologies
- Python
- Pandas
- Statsmodels
- Streamlit

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Dataset
Johns Hopkins COVID-19 dataset
